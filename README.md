# Campus Navigation API

A Flask-based campus navigation system that helps students find the shortest route between campus locations using Dijkstra's Algorithm.

## Features

- Shortest path calculation
- Graph-based routing
- REST API support
- GET, POST, PUT, DELETE operations
- HTML/CSS frontend
- JSON-based campus map

## Technologies

- Python
- Flask
- HTML
- CSS
- JSON
- Dijkstra's Algorithm
- Postman
- GitHub

## Run

pip install flask

python app.py

http://127.0.0.1:5000
